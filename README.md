## Configuration
No additional configuration is required. The module automatically creates the Exhibition Info Block, which can be placed in any region through the Drupal block layout interface.

## Usage
The Exhibition Info Block will automatically display on 'exhibition' node pages where the `field_typ` references a taxonomy term with ID 3. The block provides the following information:

- Exhibition date
- Taxonomy terms
- Author(s)
- Related person(s)
- Exhibition description
- Call-to-action button
- Sold out status
- Free exhibition status
- Additional information
- Opening hours
- Downloadable files
- Related content
- Exhibition gallery
- Video gallery
